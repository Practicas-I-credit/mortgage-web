# Hipoteca In Spain

Web de asesoría hipotecaria especializada en compradores estadounidenses en España.

🌐 Próximamente en producción

## Stack

- React 19 + Vite
- React Router DOM
- CSS Variables

## Instalación
- npm install
- npm run dev

## Estructura

src/
pages/        # Home, Services, About, Contact, Privacy, NotFound
components/   # Header, NavBar, Footer
docs/           # registro de progreso y documentación
public/images/  # imágenes y logo

## Páginas

| Ruta            | Página    |
|-----------------|-----------|
| `/`             | Home      |
| `/services`     | Servicios |
| `/about`        | Nosotros  |
| `/contact`      | Contacto  |
| `/*`            | 404       |
